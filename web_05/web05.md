연결 후, 라이브러리 내 css, js 파일과 class명으로 연결하여 사용

## form
### <form>은 데이터를 서버에 제출하기 위해 사용하는 태그
- form의 속성:
    - action: 서버를 처리할 URL
    - method: 서버 제출 시 사용할, HTTP 메서드

## input
### <input>은 사용자로부터 데이터를 입력받기 위한 태그
- input의 속성:
    - name: form control 시에 사용되는 "이름"
    - valued: form control에 적용되는 값
